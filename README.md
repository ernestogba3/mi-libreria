# 📚 Mi Estantería Virtual

Aplicación web para gestionar los libros que voy leyendo.
Construida con Node.js, Express, SQLite y EJS siguiendo arquitectura MVC.

## Tecnologías
- Node.js
- Express
- Better-SQLite3
- EJS
- Method-Override

## Instalación

1. Clona el repositorio
   git clone https://github.com/tu-usuario/mi-estanteria.git

2. Entra en la carpeta
   cd mi-estanteria

3. Instala las dependencias
   npm install

4. Arranca el servidor
   node server.js

5. Abre el navegador en
   http://localhost:3000

## Funcionalidades
- Ver todos los libros en formato tarjeta
- Añadir nuevos libros
- Editar título, autor, estado y valoración
- Eliminar libros
